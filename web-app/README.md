# Talk-To-Docs Web Application

## Commands

```bash
pnpm dev        # Start Vite dev server (http://localhost:5173)
pnpm build      # Type-check + production build
pnpm lint       # ESLint
pnpm format     # Prettier (writes in-place)
pnpm preview    # Preview production build
```
